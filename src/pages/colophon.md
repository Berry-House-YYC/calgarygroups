## About this site

**Calgary Groups** is a community directory built with open-source tools and maintained by volunteers.

### Technology

- **Static site generator**: [Eleventy](https://www.11ty.dev/) (11ty)
- **CSS framework**: [Tailwind CSS](https://tailwindcss.com/)
- **JavaScript**: [Alpine.js](https://alpinejs.dev/) for interactivity
- **Icons**: [Font Awesome](https://fontawesome.com/)
- **Hosting**: [Netlify](https://www.netlify.com/)
- **Source code**: Available on [GitHub](https://github.com/Berry-House-YYC/calgarygroups)

### Design

The site features a custom brand color palette inspired by Calgary's warm community spirit:

- **Light mode**: Cream and seashell backgrounds with pumpkin spice and dark cyan accents
- **Dark mode**: Coffee and slate backgrounds with bright cyan highlights
- **Typography**: System font stack for optimal performance
- **Progressive Web App**: Installable on mobile and desktop devices

### Accessibility

This site is designed to be accessible and inclusive:

- WCAG 2.1 AA compliant
- Keyboard navigation support
- Screen reader friendly
- High contrast color ratios
- Responsive mobile design

### Privacy

- No tracking cookies
- No analytics (currently)
- No third-party advertising
- Forms submitted via Netlify Forms (data stored by Netlify)
- Service worker caches pages locally for offline access
